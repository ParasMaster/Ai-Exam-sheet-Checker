AI-Powered Answer-Sheet Grading System

Automating assessment with advanced OCR, NLP, and human-in-the-loop for accurate, scalable grading.

Previous Model Overview

The first prototypes used Tesseract OCR and exact keyword matching. Accuracy was limited (~70%), and paraphrased answers weren’t recognized.

• Basic OCR (printed only)
• Keyword matching
• Manual flagging
• No semantic analysis

Current Project Plan

Next-gen system integrates Google Vision, transformer-based semantic matching, confidence scoring, and human review workflows.

• Hybrid OCR+HTR
• Semantic embeddings
• Confidence thresholds
• Human-in-loop QC

Achievements & Metrics

Prototype v0.3 achieved 85% grading accuracy and cut grading time by 50% compared to manual evaluation.

• 85% accuracy
• 50% time saved
• Synonym support
• Early confidence scoring

Benefits & Impact

Reduces educator workload by 80%, delivers instant feedback, and enables large-scale exams with consistent, unbiased grading.

• −80% grading time
• Instant feedback
• Bias reduction
• Analytics dashboard
  • Frees teachers from repetitive tasks
  • Supports remote and inclusive assessments
  • Generates analytics for pedagogical insights
  • Math formula OCR and derivation checking
  • Diagram recognition for sciences
  • Integration with LMS and proctoring
  • Peer learning and predictive interventions
