AI-Powered Answer-Sheet Grading System

Comprehensive Project Report

  • Basic OCR Integration using open-source Tesseract for printed text only.
  • Keyword Matching for short-answer grading.
  • Manual Workflow for flagged answers.
  • Limitations: poor handwriting accuracy (~70%), no semantic analysis, no confidence scoring.

2.1 Architecture

  1. Client UI: file uploads, status dashboard
  2. Backend Services:
    • OCR microservice (Google Vision + custom HTR)
    • Semantic analysis (transformers)
    • Grading engine with confidence thresholds
    • Manual-review queue
  3. Data Storage: MongoDB for documents, PostgreSQL for grades/logs
  4. DevOps: Docker/Kubernetes, RabbitMQ, auto-scaling
