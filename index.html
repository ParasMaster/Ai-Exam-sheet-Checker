AI-Powered Answer-Sheet Grading System

Empowering education through innovation: An AI-driven platform that intelligently processes handwritten and typed answers, provides semantic scoring with high precision, and offers real-time, personalized feedback while keeping educators in control for ultimate reliability.

Prototype 1: Building the Basics

Our journey started with a simple yet effective prototype using Tesseract OCR focused on printed text and straightforward keyword matching. It set the foundation by automating basic grading tasks, achieving around 68% accuracy in controlled tests, though it struggled with variations like handwriting or synonyms.

Quick Facts

  • OCR Engine: Tesseract (optimized for printed text)
  • Matching Method: Rule-based keywords with exact matches
  • Accuracy Level: Approximately 68%
  • Key Limitation: Limited to printed inputs; no support for semantic variations

Version 2: Introducing Semantics

Building on the first prototype, we integrated Google Vision OCR to handle both printed and handwritten text, paired with BERT embeddings for semantic understanding. This upgrade allowed for partial credit assignment and synonym recognition, improving accuracy to about 82% and reducing the need for manual reviews by 40%.

Quick Facts

  • OCR Engine: Google Vision with HTR support
  • Matching Method: BERT-based semantic embeddings
  • Accuracy Level: Approximately 82%
  • Efficiency Gain: 40% fewer manual review flags

Current System: Advanced Hybrid Model

The latest iteration features a robust pipeline designed for real-world scalability:

  • Enhanced OCR/HTR using Google Vision and EasyOCR, delivering up to 95% word recognition accuracy even on challenging handwritten inputs
  • Semantic matching powered by RoBERTa transformers calibrated with GPT-4, achieving an F1 score of 0.93 for precise scoring
  • Adaptive confidence scoring that flags only 15% of cases for human review, ensuring efficiency without sacrificing quality
  • Human-in-the-loop oversight for complex or ambiguous responses, blending AI speed with human insight

Overall, this system provides 88–92% automated accuracy and shaves off about 75% of traditional grading time, making it a game-changer for busy educators.

Highlights

Bias Reduction & Fairness Features

To promote equity, the system includes anonymous grading (no student identifiers visible) and dynamic rubrics that apply changes retroactively. This leads to more consistent outcomes, with studies showing reduced bias in AI-assisted vs. fully manual grading.

Fairness Insights

  • Anonymous Processing: Prevents unconscious bias
  • Uniform Rubrics: Ensures consistency across all submissions
  • Bias Metrics: Built-in checks for equitable scoring
